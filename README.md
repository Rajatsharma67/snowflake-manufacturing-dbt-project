# snowflake-manufacturing-dbt-project
A manufacturing company want to analyze:
- Production output
- Inventory levels
- Machine efficiency
- Delays in production


# Manufacturing Analytics (Snowflake + dbt)

## Overview
End-to-end manufacturing analytics pipeline using dbt and Snowflake.

## Features
- Incremental fact tables
- Machine efficiency calculation
- Inventory classification
- SCD snapshot for inventory tracking

## Tech Stack
- Snowflake
- dbt

## Run Steps
dbt seed
dbt run
dbt test
dbt snapshot
